>   -o <file>                Place the output into <file>.

なるほど、"$ gcc test.c -o test"はtest.c -oをコンパイルして、その結果のファイルをtestとして作っていると予想した。

それで、"$ ./test"で実行することになるのか。

```

         printf("Hello, World.\n");
return 0;}
```
